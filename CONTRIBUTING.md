# Contributing

欢迎基于这个项目继续定制你自己的 AI 人格系统。

## 建议贡献方向

- 更稳的人格模板
- 更细的 HEARTBEAT 策略
- 更好的 memory 模板
- 更通用的 USER 模板
- 面向不同使用场景的安装说明

## 提交建议

1. 先 fork 仓库
2. 新建分支
3. 保持修改聚焦
4. 避免提交私人聊天记录、密钥、平台 ID、日志或本地工作区内容
5. 提交前检查 README 和示例是否仍然一致

## 风格建议

- 优先结构清晰、可复用
- 个人化内容与通用模板尽量分离
- 对安全边界保持明确说明
- 尽量给出可执行、可迁移的配置思路
